# Customer Subscription Churn Analysis

## Business Context
This project analyzes customer subscription data to understand churn behavior and identify opportunities to improve customer retention.

## Objectives
- Measure churn and retention trends
- Analyze customer usage behavior
- Identify key factors associated with customer churn
- Provide data-driven recommendations

## Dataset
Public subscription dataset including customer plans, usage behavior, and churn indicators.

## Tools
- SQL
- Python (Pandas, Matplotlib)
- Tableau / Power BI

## Key Questions
- How does churn vary by subscription plan?
- What usage patterns are associated with churn?
- Are payment or support issues linked to churn?

## Status
- SQL analysis completed. Documentation will be updated after EDA and dashboarding.
